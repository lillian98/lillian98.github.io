状态伪类,建议和transition结合

  1. 动态伪类

    :hover用户把鼠标移动到元素上面时触发

    .state-hover img:hover{-webkit-box-reflect: below 0px -webkit-gradient(linear, left top, left bottom, from(transparent), color-stop(.7, transparent), to(rgba(255,255,255,0.4)));box-reflect: below 0px -webkit-gradient(linear, left top, left bottom, from(transparent), color-stop(.7, transparent), to(rgba(255,255,255,0.4)));box-shadow: 0px 0px 20px rgba(0,0,0,0.8); }

    :active用户点击元素那一下的效果（正发生在点的那一下，松开鼠标左键此动作也就完成了）

    按住我加能量
    .sec-state .a-active:active .a-active-energy{background-color: #cf0000;height: 100%;}

    :focus元素成为焦点（表单比较常用）

  2. :target当目标元素被定位为当前文档位置时生效的样式 #secelements:target{display: block}//页面链接为url+#secelements时，显示为block

  3. :checked（ie6-ie8不支持），仅对radio、checkbox的input元素有效，建议使用时和其他标签结合设置样式

    input-check:checked + label{background-color:#1f3c3f;}

  4. :disabled,:enabled（ie6-ie8不支持）

    注意与html中disable属性区分,js中disable属性

    .sec-state input:disabled{background-color: #ffc;}

    http://www.456bereastreet.com/lab/styling-form-controls-revisited/disabled/

    5. 下述伪类ie10+支持（对input支持较好），具体可见：https://developer.mozilla.org/en-US/docs/Web/CSS/:invalid

    和属性选择器或before、after伪类结合用提高用户体验.input new type:color、date、datetime、datetime-local、month、week、time、email、number、range、search、tel、url

  5. :valid应用于有效元素：

    :invalid应用于空的必填元素，以及未能与type或pattern属性所定义的需求相匹配的元素：

    :required应用于具有必填属性required的表单控件；

    :optional应用于没有必填属性required的所有表单控件



    :required{background-color:#ffdddd;}
    input:valid{background-color: #ddffdd;}
    input:invalid {background-color: #ffdddd;}
    input:required {border-color: #800000;border-width: 3px;}

  6. :in-range应用于具有范围限制的元素，其中该值位于限制内；比如具有min和max属性的number和range输入框；

    :out-of-range与:in-range选择相反，其中该值在限制范围外；

    date

    input:out-of-range { border: 1px solid #CB000F; color: red } input:in-range { color: #00882D; font-weight: bold; }

  7. :read-only应用于其内容无法供用户修改的元素；

    :read-write应用于其内容可供用户修改的元素，比如输入框；

    目前浏览器支持不完善，暂不建议使用

结构伪类选择器Structural Pseudo-Classes（http://www.w3cplus.com/css3/pseudo-class-selector）

  1. first-child(选择某个元素的第一个子元素)
  2. last-child(选择某个元素的最后一个子元素)
  3. nth-child(odd)奇数匹配，nth-child(even)偶数匹配
  4. nth-child(number)匹配第number个
  5. nth-child(an)匹配为a倍数的元素，nth-child(an+b)，exp：nth-child(-1n+8)匹配前8个
  6. nth-last-of-type(n)倒数第n个元素（按父元素下来算，不分当前标签类） first-of-type第一个
  7. only-of-type匹配属于同类型中唯一兄弟元素的E

属性选择器 attribute selector

  • E[attr]该选择器定位具有属性attr的任何元素E：
  • E[attr=val]该选择器定位具有属性attr且属性值为val的任何元素E：
  • E[attr|=avl]该选择器定位具有属性attr且属性值为val或以val-开始的任何元素E：
  • E[attr~=val]该选择器定位具有属性attr且属性值为完整单词 val 的任何元素E：
  • E[attr^=val]该选择器定位具有属性attr且属性值以val开头的任何元素E：
  • E[attr$=val]该选择器与E[attr^=val]正好相反，定位具有属性attr且属性值以val结尾的任何元素E：
  • E[attr*=val]该选择器与E[attr~=val]相似，但更进一步，定位具有属性attr且属性值任意位置包含val的元素E，val可以是一个完整的单词，也可以是一个单词中的一部分：

其他选择器

  1. :empty选择内容为空的某元素,用于调试可视化过滤空元素

    div:empty, li:empty, p:empty, td:empty, th:empty{padding:1em;background-color:#000 !important;color:#fff}
  2. not(s)否定

  3. ~兄弟元素选择器

    #div9:hover ~ div:nth-child(9n) { background: yellow; }

  4. :root根元素，始终指html元素；和html类似，但优先级高于html

    html{background-color:#ccc;}
    :root{background-color:#eee;}
